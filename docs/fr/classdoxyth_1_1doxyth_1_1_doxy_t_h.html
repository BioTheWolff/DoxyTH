Description détaillée

Classe automatiquement appelée par Doxygen comme un processeur de fichier.

Ce fichier (donc cette classe) est automatiquement appelée par Doxygen pour qu'elle travaille sur un fichier qui suit la règle définie dans l'option de configuration FILE_PATTERNS. Par défaut, DoxyTH mets la valeur de FILE_PATTERNS à "*py=.dthb", ".dthb" étant le fichier batch appelé par Doxygen quand il rencontre un fichier se terminant par ".py".

Le fichier batch contient "python -m doxyth.doxyth <lang> %1", LANG étant la langue actuelle, automatiquement changée par la classe Gendoc chaque fois qu'on change de langue.

Le fichier dont le chemin est donné est alors lu, et chaque doc_id référencé et toutes les lignes de documentation où le doc_id a été trouvé sont remplacées par la documentation liée à cet identifiant.

Le fichier est ensuite écrit dans la console ligne par ligne pour que Doxygen le lise et le process. Si un post-processus a été défini, les lignes du fichier, plutôt que d'être écrites, sont données au post-processus qui s'occupera de les écrire dans la console après avoir fait son travail sur les lignes.
